# Game-programming

A repository for game programming learning and practice, covering core game development technologies, prototype implementations, and algorithm practice.

---

## 🌾 Rural Magic RPG Prototype
A cozy low-fantasy RPG prototype set in a quiet village, blending everyday rural life with gentle magic.  
The core focus includes:
- Basic 3D scene construction and lighting
- Simple player movement and camera control
- Basic interaction logic and scene management
- Unity workflow practice with both custom and pre-built environments

### Scene Paths
- `Assets/Scenes/New Scene.unity` — Main gameplay scene (custom rural magic RPG setting)
- `Assets/Scenes/Suntail Village.unity` — Training/practice scene using the Suntail Village demo environment
